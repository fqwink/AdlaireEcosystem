# Adlaire Ecosystem

Adlaire Ecosystemは`v0.001`からBaaS Projectとしてゼロベースで再スタートします。

`v0.001`で計画する中核機能はDeployment SystemとRealtime Databaseです。

Realtime DatabaseのDatabaseはSQLiteを正選定します。

## 主要ディレクトリ

```text
Core/
Applications/
docs/
tests/
```

## テスト

```sh
php tests/debug.php
```

詳細仕様と設計判断は`ADLAIRE-ECOSYSTEM.md`へ集約します。
